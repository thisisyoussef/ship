# Submission Bundles

Use this directory for assembled output bundles that are meant for review, demo, or submission.

## Put Here

- demo scripts prepared for reviewers
- packaged evidence bundles
- final submission notes
- review-oriented summaries that combine material from several plans or stories

## Do Not Put Here

- working implementation specs
- active story contracts
- scratch planning notes

Those belong in `docs/plans/` or `docs/user-stories/`.
